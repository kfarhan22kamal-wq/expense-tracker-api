# Expense Tracker API

A simple REST API built using Flask to manage and track expenses.

## Features
- Add expenses
- View all expenses
- Delete expense
- Calculate total spending
- Input validation and error handling

## Tech Stack
- Python
- Flask

## How to Run
pip install flask  
python app.py  

## API Endpoints
POST /expenses  
GET /expenses  
DELETE /expenses/<id>  
GET /expenses/total
